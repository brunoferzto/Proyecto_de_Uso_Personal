
FASES - SUBFASES - EQUIPOS

Las Fases tienen una lista de Subfase 
Las Subfases tienen una lista de Integrantes y una lista de Reglas por Posición.
 * De esta forma es posible crear una Fase con distintios tipos, ej (Grupo A 8 integrantes, Grupo B 4 integrantes)
	Se decidio finalmente que Reglas x Posición sean una lista en Fases que se aplicara a todas las subfases menos 
	  a las que tengan su atributo en false (bool RespetaReglas) es posible también crear una Fase con distintios tipos.
	     Motivo  *Habra un tipo de reglas por posición que se aplicara generalmente, para no repetir esta lista en todas las Subfases

modelo : Objeto 'Fase' tiene List<'Subfase'>
                  Objeto 'Subfase' tiene List<Integrantes>
                   Objeto Integrante tiene 1 Equipo.
	 Objeto Equipo tiene 1 Competidor
	  Objeto Competidor es el final de la creación de una Subfase.

A LA BD:
1- Se va a la BD a Crear Fase
2- Se traen todas las Subfases
3- Se traen los integrantes de esta Subfase
4 - Se busca el Equipo para este Integrante
5 - Se busca el competidor para este Equipo
